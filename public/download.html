  1. Prerequisites

    Install Node.js (v18 or later recommended). This includes npm, the package manager used to install dependencies.

  2. Clone the repository

    Open your terminal and run:

    git clone https://github.com/hena-lee/velvet-tarot.git

  3. Install dependencies

    Navigate into the project folder and install the required packages:

    cd velvet-tarot && npm install

  4. Get a Gemini API key

    Visit Google AI Studio and create a free API key. You'll need a Google account.

  5. Configure your API key

    Create a file called .env in the project root with your key:

    GEMINI_API_KEY=your_key_here

  6. Start the app

    Launch the server, then open the app in your browser:

    npm start

    Then visit http://localhost:3000
